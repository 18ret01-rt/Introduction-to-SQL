# Introduction-to-SQL
Items found within this repository are for the MEDB-5508 course which gives a brief overview to Structured Query Language (SQL)


The data for these assignments can be obtained from the MIMIC III database which is publically available.

Files found within the src folder contain both HTML and QMD versions of the promgamming assignments. They are all under the public domain so please use them as you wish. 

Lastly, files found within the results folder contain PDF copies of the R Quarto programs to run the SQL queries to answer the required questions in the MEDB-5508 coursework. 

This course was taken as MEDB-5530 (independent study) but was replicated of the traditional MEDB-5508 course.
